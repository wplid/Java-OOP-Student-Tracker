# Java OOP Student Tracker 🎓

A simplified Java-based application that applies Object-Oriented Programming (OOP) concepts to manage student data.

## 📝 Description
This project is a mini software application written in Java aimed at managing student records. The project consists of two main classes: 
* The `Etudiant` class, which represents the student data structure.
* The `Main` class, which contains the program to execute and test the operations.

## ✨ Features
* **Encapsulation:** Protects student data and uses methods to get and modify data (Getters & Setters).
* **Data Validation:**
  * Ensures the student's age is valid (over 18).
  * Verifies that the Baccalaureate average (`moyBac`) is logical (between 0 and 20).
  * Ensures the phone number consists of exactly 10 valid digits.
* **Array Management:** Uses an array to store data for multiple students and manage them programmatically.
* **Data Updating:** Includes the ability to automatically upgrade the study level (`niveau`) for all students at once using loops.

## 💻 Technologies Used
* Java (Standard Edition)
* Object-Oriented Programming (OOP)
* Eclipse IDE
